# ActionGame

# Control
- W = 앞으로 이동
- A = 왼쪽으로 이동
- S = 뒤로 이동
- D = 오른쪽으로 이동
- R = 장전
- Space = 점프 및 도약

# Information
1. 5스테이지 마다 보스전이 진행됩니다.
2. 몬스터를 잡으면 코인을 얻을 수 있습니다.
3. 점수는 게임을 재시작해도 저장됩니다.

# How to play video games
- 아이템과 장비를 구매해서 스테이지를 높여가세요.
